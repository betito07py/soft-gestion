# Soft_Gestion

Sistema de gestión desarrollado en VB.NET y SQL Server.

## Tecnologías
- Visual Studio 2026
- VB.NET
- SQL Server
- Git / GitHub

## Estructura del repositorio

- `src/` → código fuente de la solución
- `database/` → scripts SQL versionados
- `docs/` → documentación funcional y técnica
- `tools/` → scripts auxiliares y utilidades

## Convención de base de datos

- `schema/` → tablas, claves, índices
- `migrations/` → scripts de cambio incremental
- `stored_procedures/` → procedimientos almacenados
- `views/` → vistas
- `functions/` → funciones
- `jobs/` → scripts de SQL Agent u otros procesos
